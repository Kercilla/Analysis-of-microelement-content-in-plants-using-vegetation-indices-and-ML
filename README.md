##### Цель работы #####

Основная цель данной работы заключается в исследовании и разработке подхода к неинвазивному определению содержания микроэлементов в растениях. Для достижения этой цели необходимо выполнить следующие задачи:
    1. Провести анализ данных для выявления корреляций между содержанием различных микроэлементов (медь, железо, марганец, цинк) в растениях и вегетационными индексами (NDVI, RVI, DVI, SAVI, MSAVI2), полученными с помощью дистанционного зондирования.
    2. На основе полученных результатов и анализа литературных данных предложить вариант реализации системы, использующей технологии машинного обучения и нейронных сетей, для определения количества того или иного микроэлемента в растениях по их изображениям.
